## 포켓몬 도감 URI

| 메소드 | 경로                   | 설명                  |
| ------ | ---------------------- | --------------------- |
| GET    | /pokemons              | 포켓몬 전체 조회      |
| GET    | /pokemons/{pokemonIdx} | 특정 포켓몬 정보 조회 |
| POST   | /pokemons              | 포켓몬 등록           |
| PUT    | /pokemons/{pokemonIdx} | 포켓몬 정보 수정      |
| DELETE | /pokemons/{pokemonIdx} | 포켓몬 삭제           |
